工具箱

简介

记录我平时开发时使用较多的快捷键和命令。

IDEA-hotkey

格式化代码：command + option + L

大小写切换：command + shift + U

批量注释：command + shift + /

删除光标所在行：command + DEL

复制多行：command + D

移动当前代码行：option + shift + 上下键

set/get：control + enter

补齐分号：command + shift + enter

快速修复+导包：option + enter

代码补齐：option + /

显示类的结构：command + 7

代码环绕：command + option + T

macOS-commandline

删除当前行：control + U

清屏：control + L

显示当前文件路径：pwd

显示出隐藏的文件夹：command + shift + .

MySQL

Start MySQL Server or sudo /usr/local/mysql/support-files/mysql.server start

mysql -uroot -p

Redis

redis-server

redis-cli

redis-cli -h 127.0.0.1 -p 6379 shutdown

Kafka

cd kafka_2.12-2.4.1/

bin/zookeeper-server-start.sh config/zookeeper.properties // 先开 ZooKeeper，指定配置文件

bin/kafka-server-start.sh config/server.properties // 再开 Kafka 的 server，同样指定配置文件

Elasticsearch

elasticsearch 启动服务端 localhost:9200

cd ~/elasticsearch-head

npm run start 启动客户端 localhost:9100

客户端可以用 Postman 代替
